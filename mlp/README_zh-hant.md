更好的網上香港大學體驗，與SuperEZ

## 設置
1. 從Google Chrome網上應用店安裝最新版本
2. 前往任何香港大學服務頁面
3. 將鼠標移到右下角
4. 點擊設置圖標
5. 設置您的偏好
6. 享受吧！

有關詳細文檔，請訪問我們的[維基](https://github.com/EZ-HKU/SuperEZ/wiki)。

### FastHKU
一步訪問香港大學服務
### mO.odle
讓Moodle重回輝煌！
### SuperLoad
一鍵下載所有課程文件！
