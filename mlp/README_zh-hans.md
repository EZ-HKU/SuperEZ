更好的在线香港大学体验，尽在SuperEZ

## 设置
1. 从Google Chrome网上应用店安装最新版本
2. 访问任何香港大学服务页面
3. 将鼠标移动到右下角
4. 点击设置图标
5. 设置您的偏好
6. 享受吧！
   
## 维基
有关详细文档，请访问我们的[维基](https://github.com/EZ-HKU/SuperEZ/wiki)。

### FastHKU
一步访问香港大学服务
### mO.odle

### SuperLoad
