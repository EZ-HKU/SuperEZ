## Cài Đặt
1. Cài đặt phiên bản mới nhất từ Google Chrome Web Store
2. Truy cập bất kỳ trang dịch vụ HKU nào
3. Di chuyển chuột đến góc dưới bên phải
4. Nhấp vào biểu tượng Cài đặt
5. Thiết lập sở thích của bạn
6. Tận hưởng!
